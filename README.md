# JewelBill 💎

Jewellery GST Invoicing application designed for wholesalers. Efficiently manage company settings, customer details, and generate professional GST-compliant invoices with automatic tax calculations.Open source project. 

## 🚀 Live Demo
**Access the hosted application here:**  
[https://shreedhardev.github.io/Jewel_Billing_App/](https://shreedhardev.github.io/Jewel_Billing_App/)

---

## ✨ Features

- **Secure Login:** Professional authentication interface for wholesalers.
- **Dynamic Invoicing:** 
  - Real-time calculation of Gold rates and Making charges.
  - Automatic GST breakdown (CGST/SGST or IGST).
  - Discount handling and Grand Total rounding.
- **Professional Layout:** Premium design with interactive previews.
- **Export Options:** 
  - Download as high-quality PDF.
  - Direct print optimization.
- **Client Management:** Store customer GST and address details for each bill.
